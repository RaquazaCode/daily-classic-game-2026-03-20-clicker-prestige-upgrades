## How To Play
- Press `Enter` on title to start a run.
- Click the large `CLICK ZONE` panel or press `Enter` in-run for active income.
- Buy upgrades: `C` for Cursor (click power), `F` for Factory (idle income).
- Trigger prestige with `K` once total earned coins reach 1000.
- Pause/resume with `P`; hard reset to title (including prestige) with `R`.

## Rules
- Coin gains are deterministic and tick at a fixed 100ms step.
- Upgrades require current-run coins and scale in cost after each purchase.
- Prestige requires at least 1000 total earned coins in the current run.
- Prestige grants shards, increases permanent multiplier, and resets run upgrades/coins.
- `P` freezes all simulation updates while paused.

## Scoring
- Score metric is total coins generated.
- Click income: `clickPower * prestigeMultiplier`.
- Idle income: `autoRatePerSecond * prestigeMultiplier` over deterministic ticks.
- Prestige shard award: `floor(totalCoinsEarned / 1000)`.

## Twist
- **Prestige upgrades**: each shard increases permanent multiplier by `+0.2`.
- Prestige creates a clean run reset while preserving shard-based progression.
